1. Baixe `APLICAR_CORRECOES_V47_13.py` pelo GitHub.
2. Coloque o arquivo na raiz do projeto, no mesmo nível da pasta `sistema`.
3. Execute:

```powershell
python APLICAR_CORRECOES_V47_13.py
```
